## 📌 What is OSS Pulse?

We track **GitHub's most influential repositories** - the top projects by stars, forks, community engagement, and trending topics (especially AI/ML). This isn't a random sample; it's the **elite of open source**.

**Selection Criteria:**
- 🌟 Top repositories by star count
- 🍴 Most forked projects (500+ forks)
- 🤝 Active community projects (help-wanted issues)
- 👀 Highly watched repositories (100+ watchers)
- 🤖 Trending AI/ML projects

## About

**OSS Pulse** provides daily analytics on trending GitHub repositories, offering insights into:

- 🔥 **Viral Detection** - Repos with explosive growth
- 📊 **Health Scores** - CHAOSS-inspired project health grades
- 🏢 **Organization Rankings** - Top open source contributors
- 💻 **Language Trends** - Technology adoption patterns

Data is collected daily and analyzed using statistical methods to identify patterns and trends.
