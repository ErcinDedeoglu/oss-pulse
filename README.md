## 📌 Momentum Score (0-100)

A single score that answers: **"Is this project accelerating right now?"**

| Component | Weight | What It Measures |
|-----------|--------|------------------|
| Velocity | 40% | Absolute 7-day star gain (percentile-scaled) |
| Relative Growth | 25% | Gain as % of existing stars |
| Acceleration | 25% | This week vs last week |
| Consistency | 10% | Positive growth across 7/14/30-day windows |

Viral repos (>5x baseline spike) get a +10 bonus. [Full methodology →](docs/methodology.md)

## About

**OSS Pulse Momentum Dashboard** focuses on **change rather than just current state**:

- ⚡ **Momentum Score** - Single 0-100 score combining velocity, relative growth, acceleration, and consistency
- 🔢 **Quadrant View** - Repos classified into Hot Giants, Rising Stars, Coasting Giants, and Emerging
- 📈 **Sparklines** - 7-day text-based star trajectory right next to repo names
- 🔥 **Viral Detection** - Only genuine 5x+ baseline spikes (no absolute fallback)
